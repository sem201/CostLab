# 3. Key Features (주요 기능)

- **장바구니**:

  - 상세 페이지에서 장바구니에 물건을 담을 수 있습니다.
  - 특정 상품을 삭제할 수 있습니다.

- **총 금액 계산기**:
  - 장바구니에 담은 물품들의 총 금액을 계산해줍니다..
  - checkbox를 이용하여 특정 요소만의 가격을 측정할 수 있습니다..
